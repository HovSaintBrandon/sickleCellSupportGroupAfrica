# Sickle Cell Support Group Africa

This is the official website for the Sickle Cell Support Group Africa. It is a modern, responsive, and engaging platform designed to raise awareness, provide support, and connect individuals affected by sickle cell disease across the African continent.

## Features

- **Modern Design**: Clean, vibrant, and user-friendly interface.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices.
- **Interactive Elements**: Smooth scrolling, hover effects, and dynamic content.
- **Key Sections**:
    - **Hero Section**: Engaging introduction with a powerful call to action.
    - **About Section**: Detailed information about Sickle Cell Disease.
    - **Our Impact**: Showcasing the organization's reach and achievements.
    - **Stories**: Personal testimonials and success stories.
    - **Get Involved**: Ways for visitors to contribute and help.
    - **Blog**: Latest news and articles.
    - **Events**: Upcoming and past events.
    - **Contact**: Easy ways to get in touch.
- **Navigation**: Intuitive desktop and mobile navigation.
- **Visuals**: High-quality icons and modern typography.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd sickleCellSupportGroupAfrica
    ```

2.  Open `index.html` in your web browser:
    ```bash
    open index.html
    ```

## Technologies Used

- **HTML5**: Semantic markup for the website structure.
- **CSS3**: Modern styling with custom properties and animations.
- **JavaScript (Vanilla)**: Interactive features and DOM manipulation.
- **Lucide Icons**: Lightweight and beautiful icon library.

## Project Structure

## Development

### Adding New Content

- **Pages**: Create new HTML files in the root directory or add new sections to `index.html`.
- **Styles**: Add custom CSS rules to `assets/css/main.css`.
- **Scripts**: Add or modify JavaScript functionality in `assets/js/main.js`.

### Running Locally

To run the website locally with a development server, you can use the built-in Python HTTP server:

```bash
cd sickleCellSupportGroupAfrica
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.
